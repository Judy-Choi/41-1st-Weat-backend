# 🥩 Weat
안녕하세요, 저희는 'Weat' 팀입니다.
- 'Weat' 은 Wecode 코딩 부트캠프 41기 학생들로 구성된 첫번째 프로젝트 팀명이자 저희의 서비스 이름입니다.
- '[정육각](https://www.jeongyookgak.com/index)' 사이트 클론 코딩
    - '정육각' 은 한국의 스타트업이 런칭한 서비스로 ‘초신선’ 한 식료품, 특히 '육류' 를 주로 판매합니다.

🖥️ [Demo video](https://youtu.be/Es1Xk93CRZo)
